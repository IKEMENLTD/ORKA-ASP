01 / SERVICE

サービスについて

ORKA ASPとは何ですか？

+

ORKA ASPは、完全招待制のクローズドアフィリエイトネットワークです。厳選された広告主と高品質なアフィリエイターのみが参加できる、信頼性の高いプラットフォームを提供しています。

一般的なオープンASPとは異なり、参加者を限定することで、不正行為の防止、高単価案件の提供、専任サポートの実現を可能にしています。

参加するにはどうすればいいですか？

+

ORKA ASPへの参加方法は以下の通りです：

  • 既存メンバーからの紹介
  • 公式サイトからの審査申請
  • スカウト（実績のあるアフィリエイター対象）

審査では、過去の実績、運営メディアの品質、アフィリエイト活動の方針などを総合的に評価します。

02 / FEE

報酬・料金について

報酬はいつ支払われますか？

+

ORKA ASPでは、月2回の支払いサイクルを採用しています：

  • 前半締め（15日）→ 当月末払い
  • 後半締め（末日）→ 翌月15日払い

最短15日で報酬を受け取ることができます。振込手数料は当社負担です。

最低支払額はありますか？

+

最低支払額は 1,000円 からとなっています。未達の場合は次回支払いに繰り越されます。

月間報酬が10万円を超える場合は、週払いオプションもご利用いただけます。

広告主として参加する場合の費用は？

+

広告主様向けには3つのプランをご用意しています：

BASIC 初期費用 0円 成果報酬 + 手数料30%
STANDARD 月額 50,000円 成果報酬 + 手数料20%
ENTERPRISE 月額 200,000円 成果報酬 + 手数料10%
03 / SYSTEM

システムについて

トラッキングの仕組みを教えてください

+

ORKA ASPでは、最新のトラッキング技術を採用しています：

  • Cookie + フィンガープリント：複合トラッキングで取りこぼしを防止
  • サーバーサイド計測：広告ブロッカーの影響を回避
  • リアルタイム反映：成果発生から数分以内に管理画面に反映
  • クロスデバイス対応：PCとスマホをまたぐ成果も計測

API連携は可能ですか？

+

はい、REST APIを提供しています。以下の操作が可能です：

  • 成果データの取得・分析
  • リンク生成の自動化
  • レポートの自動取得
  • Webhook による成果通知

詳細は API ドキュメント をご確認ください。

成果の確定基準は何ですか？

+

成果確定の基準は案件ごとに異なりますが、一般的な流れは以下の通りです：

01 成果発生（クリック/申込）
02 広告主側で審査
03 成果確定（15-60日）

専任担当者はつきますか？

+

はい、以下の条件で専任担当者がアサインされます：

  • アフィリエイター：月間報酬10万円以上
  • 広告主：全プランで専任担当者付き

専任担当者は、案件提案、最適化アドバイス、特別単価交渉などをサポートします。

セミナーや勉強会はありますか？

+

定期的にメンバー限定のイベントを開催しています：

  • 月次ウェビナー：最新トレンド、成功事例の共有
  • 四半期ミートアップ：東京・大阪・福岡で開催
  • 年次カンファレンス：業界トップの登壇者を招いた大型イベント

これらのイベントはすべて参加無料です。
