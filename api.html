01 / FEATURES

API機能

RESTful API

標準的なREST設計で直感的に利用可能。JSON形式でのデータ交換に対応。

Webhook

リアルタイムイベント通知。成果発生、承認、否認を即座に通知。

OAuth 2.0

セキュアな認証。APIキー＋シークレットによる強固な認証。

Bulk API

大量データの一括処理。CSV/JSONでのエクスポート・インポート対応。

03 / EXAMPLE

コードサンプル

curl -X GET "https://api.orka-asp.com/v2/reports/conversions" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "start_date": "2025-01-01",
    "end_date": "2025-01-31",
    "status": "approved"
  }'
const response = await fetch(
  'https://api.orka-asp.com/v2/reports/conversions',
  {
    method: 'GET',
    headers: {
      'Authorization': `Bearer ${API_KEY}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      start_date: '2025-01-01',
      end_date: '2025-01-31',
      status: 'approved'
    })
  }
);

const data = await response.json();
import requests

response = requests.get(
    'https://api.orka-asp.com/v2/reports/conversions',
    headers={
        'Authorization': f'Bearer {API_KEY}',
        'Content-Type': 'application/json'
    },
    json={
        'start_date': '2025-01-01',
        'end_date': '2025-01-31',
        'status': 'approved'
    }
)

data = response.json()
